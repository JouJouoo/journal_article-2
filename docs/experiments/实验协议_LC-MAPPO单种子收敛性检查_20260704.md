# LC-MAPPO 单随机种子收敛性检查实验协议

## 研究问题

在当前代码实现和默认环境配置下，`lc_mappo` 是否能够完成训练并呈现早期收敛迹象。

## 数据与标准算例

- 使用数据：`configs/default.yaml` 的默认 synthetic 场景。
- 是否为官方标准算例：否。
- 标准算例检查：本次不使用 IEEE 33/69 标准算例，因此不执行 `validate_benchmark_sources.py`。默认场景预检查项为：动作尺度、负荷尺度、线路容量、储能边界、可行性投影和指标口径均来自 `configs/default.yaml` 与现有环境实现。

## 方法范围

- 训练方法：`lc_mappo`。
- 对比方法：无。
- 消融实验：无。
- 随机种子：`7`。

## 执行顺序

1. Smoke test：`3` episodes，确认训练入口、checkpoint 和 metrics 可以生成。
2. 短训：`200` episodes，用于观察单 seed 训练曲线。
3. 评估：训练后执行 `5` evaluation episodes，使用独立 `seed_start=100000`。

## 核心指标

- 训练曲线：`mean_reward`、`total_reward`、`actor_loss`、`critic_loss`、`entropy`。
- 可行性与结算：`feasible_rate`、`settlement_success_rate`、`max_violation`。
- 低碳资产：`lccoins`、`lccoins_candidate`、`reward_eco`、`reward_coin`。

## 验收阈值

- Smoke test 正常结束，并生成 `lc_mappo_checkpoint.pt` 与 `lc_mappo_metrics.json`。
- 200-episode 短训正常结束，训练指标均为有限数值。
- 若最后 50 个 episode 的 `mean_reward` 均值不低于最初 50 个 episode 的均值，且最后 50 个 episode 的标准差不超过最初 50 个 episode 的 1.5 倍，则记为“有早期收敛迹象”。
- 若不满足上述曲线条件，只能记为“训练可运行，但本轮未显示清晰收敛迹象”。

## 允许支持的结论

本实验只允许支持当前默认 synthetic 场景下、单随机种子、短训长度内的可运行性和早期收敛迹象判断。不支持任何消融结论、方法优于基线结论、多种子稳定性结论、IEEE 33/69 标准算例外推结论或论文级显著性结论。
