# 低碳资产感知 MAPPO 全量实验协议（无消融版）

## 研究问题

在 P2P 低碳能源交易环境中，论文提出的低碳资产感知 MAPPO（LC-MAPPO；代码内部兼容键为 `tecsf` / `lc_mappo`）是否能够在结算、电网兜底与网络可行性约束下，稳定提升低碳资产、低碳贡献和系统运行质量，并在 IEEE 33/69 标准算例上保持可复现实验表现。

本轮按用户要求不运行消融实验，因此不运行 `no_lccoins`、`no_lagrange`、`no_chain`、`preset_low_carbon`、LCCoins sensitivity 等组件移除或参数消融。

## 数据与标准算例

- 默认场景：`configs/default.yaml` 中的 8 产消者、9 节点、24 时段场景，用于主对比、非消融压力验证、扩展性和结算鲁棒性检查。
- 标准算例：IEEE 33-bus 与 IEEE 69-bus。
- 标准算例检查命令：`python scripts/validate_benchmark_sources.py --case69-m data/case69.m --strict`。
- IEEE 69 使用本地 MATPOWER `data/case69.m` 作为权威来源校验；训练评价由 `scripts/run_improved_experiment_suite.py` 生成标准 profile/config。

## 对比方法

- 论文方法：LC-MAPPO（内部 key：`tecsf`）。
- 非消融基线：`constrained_mappo`、`safety_only`、`myopic_opt`、`heuristic`。
- IEEE 33/69 标准算例对比：`tecsf`、`myopic_opt`、`heuristic`。

## 随机种子

- Smoke test：`7`。
- 100 episode 短训：`7, 42`。
- 1000 episode 全量正式实验：`7, 42, 100, 2026, 3407`。

## 训练与评价轮数

- Smoke test：`3` episodes，`2` evaluation episodes。
- 短训：`100` episodes，`5` evaluation episodes。
- 全量正式实验：`1000` episodes，`20` evaluation episodes。

## 输出目录

- Smoke：`outputs/full_experiments_20260703_paper_aligned_smoke`。
- 短训：`outputs/full_experiments_20260703_paper_aligned_short100_no_ablation`。
- 全量正式实验：`outputs/full_experiments_20260703_paper_aligned_no_ablation`。
- 运行日志：`outputs/full_experiments_20260703_no_ablation_logs`。

## 核心指标

- 可行性与安全：`eval_settlement_success_rate`、`eval_max_violation`、`eval_feasible_rate`。
- 经济性：`eval_mean_reward`、`eval_system_cost`、`eval_participant_payment_cost`、`eval_p2p_transfer_payment`。
- 低碳与资产：`eval_grid_carbon_emission`、`eval_carbon_reduction`、`eval_q_lc`、`eval_lccoins`、`eval_lccoins_candidate`。
- 机制关联：`eval_agent_lccoins_q_lc_corr`、`eval_agent_lccoins_c_offset_corr`、`eval_agent_lccoins_candidate_corr`。
- 运行代价：`train_seconds`、`eval_seconds`、`total_seconds`、`peak_python_memory_mb`。

## 验收阈值

- LC-MAPPO 在主实验、非消融压力验证和标准算例中满足：
  - `eval_settlement_success_rate_mean >= 0.95`
  - `eval_max_violation_mean <= 0.1`
- 后处理必须生成：
  - 每个保留实验套件的 `summary.json`、`runs.csv` 或分组 CSV。
  - `figures/` 下独立 `png/svg/pdf` 图。
  - `statistics/paired_comparisons.json`。
  - `pareto/pareto_runs.json`。
  - `paper_figures/` 下主对比图、IEEE33/69 benchmark 图和训练曲线图。

## 允许支持的论文结论

本实验只允许支持以下结论：

- 低碳资产状态、CRRA 存量/增量效用、资产优势分解、信用分配和自适应裁剪构成的 MAPPO 机制能够在本代码实现和所列场景中稳定运行。
- 与指定非消融基线相比，LC-MAPPO 在可行性门槛内展示 LCCoins 积累、低碳贡献和交易/碳指标的综合表现。
- IEEE 33/69 结果只能作为标准算例复现实验支持，不能替代所有配电网规模和拓扑的一般性结论。
- 若统计检验没有通过 Holm 校正，不能宣称 `tecsf` 在所有指标上显著优于全部基线。
- 本轮不产生组件消融结论，不讨论去除 LCCoins、去除拉格朗日、去除链上结算等机制移除结果。

## 执行命令顺序

可直接使用编排脚本顺序执行第 3-5 步：

`python scripts/run_full_experiments_20260703.py --device cuda --jobs 3`

1. 标准算例检查：
   `python scripts/validate_benchmark_sources.py --case69-m data/case69.m --strict`
2. Smoke test：
   `python scripts/run_improved_experiment_suite.py --quick --episodes 3 --eval-episodes 2 --seeds 7 --device cuda --jobs 1 --benchmark-cases ieee33bw ieee69 --benchmark-standard-source authoritative --case69-m data/case69.m --output-dir outputs/full_experiments_20260703_paper_aligned_smoke`
3. 无消融短训：
   `python scripts/run_improved_experiment_suite.py --quick --episodes 100 --eval-episodes 5 --seeds 7 42 --device cuda --jobs 3 --benchmark-cases ieee33bw ieee69 --benchmark-standard-source authoritative --case69-m data/case69.m --skip-ablations --output-dir outputs/full_experiments_20260703_paper_aligned_short100_no_ablation`
4. 无消融全量正式实验：
   `python scripts/run_improved_experiment_suite.py --episodes 1000 --eval-episodes 20 --seeds 7 42 100 2026 3407 --device cuda --jobs 3 --benchmark-cases ieee33bw ieee69 --benchmark-standard-source authoritative --case69-m data/case69.m --skip-ablations --output-dir outputs/full_experiments_20260703_paper_aligned_no_ablation`
5. 论文图与训练曲线补充：
   `python scripts/plot_paper_figures.py outputs/full_experiments_20260703_paper_aligned_no_ablation --skip-ablations --output-dir outputs/full_experiments_20260703_paper_aligned_no_ablation/paper_figures`
   `python scripts/plot_ieee_benchmark_paper_figure.py --root outputs/full_experiments_20260703_paper_aligned_no_ablation --output-dir outputs/full_experiments_20260703_paper_aligned_no_ablation/paper_figures`
   `python scripts/plot_multiseed_training.py --run-dir outputs/full_experiments_20260703_paper_aligned_no_ablation/formal_multiseed/tecsf --output-dir outputs/full_experiments_20260703_paper_aligned_no_ablation/formal_multiseed/tecsf/figures`
